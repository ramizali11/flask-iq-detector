# IQ Quiz Web App

## Description
A simple IQ quiz application built using Flask, HTML, and CSS. 
Users answer multiple-choice questions and receive a score and IQ result.

## Features
- Multiple choice IQ questions
- Score calculation
- IQ evaluation
- Clean UI
- percentage calculation

## Project Structure
iq-quiz-app/
├── app.py
├── templates/
│   ├── index.html
│   └── result.html
├── static/
│   └── style.css
└── README.md



## How to Run
1. Install Python
2. Install Flask
   pip install flask
3. Run the app
   python app.py
